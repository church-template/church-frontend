## 제목

🚀 [기능개선][사역] 나머지 부서 상세 페이지에 보강 섹션 확대 적용

## 본문

## 📝 현재 문제점

- 부서 상세 페이지는 학생부(#78)만 보강 섹션(소개·알림 사항·주요 활동·특별 프로그램·활동 사진·초대)이 채워져 있고, 청년부·예배부·남선교회·여선교회는 이름·짧은 소개·캐치 문구·히어로 이미지만 있어 상세 페이지가 상대적으로 빈약하다.
- 섹션 순서(소개 → 활동 사진 → 주요 활동 → 특별 프로그램 → 알림 → 초대)와 밴드 리듬, 텍스트 가시성 상향(#80)은 페이지 조립부·공용 스타일이라 이미 모든 부서에 적용돼 있으나, 콘텐츠가 없는 부서는 해당 섹션이 자동으로 생략돼 학생부만큼 풍부하게 보이지 않는다.

## 🛠️ 해결 방안 / 제안 기능

- 학생부와 동일한 보강 섹션 구성을 나머지 부서(청년부·예배부·남선교회·여선교회)에도 채워, 모든 부서 상세가 일관된 흐름(소개 → 활동 사진 → 주요 활동 → 특별 프로그램 → 알림 → 초대)으로 보이도록 한다.
- 섹션 순서·밴드 리듬·컴포넌트·텍스트 스케일은 이미 구현돼 있으므로 그대로 재사용하고, 각 부서의 실제 콘텐츠(소개 문구·안내 정보·활동 내용·프로그램·초대 문구·활동 사진)만 큐레이션해 채운다.

## ⚙️ 작업 내용

- 청년부·예배부·남선교회·여선교회 각 부서에 아래 섹션 콘텐츠 작성·주입
  - 소개(제목·리드 문구), 특징 카드 3종
  - 알림 사항(담당자·모임 시간·연락처·모임 장소)
  - 주요 활동 목록
  - 특별 프로그램(이름·설명)
  - 초대(제목·본문)
- 각 부서 활동 사진 준비 및 연결(실제 사진 확보 전에는 placeholder 이미지로 시작)
- 모든 콘텐츠는 상수에서 주입(콘텐츠 하드코딩 금지), 학생부 데이터 구조와 동일하게 맞춤
- 각 부서 상세에서 섹션 순서·밴드 리듬·텍스트 가시성이 학생부와 동일하게 렌더되는지 확인
- 담당자·연락처 등 확정 전 항목은 임시 값 표기 방식을 학생부와 통일(미확정 시 해당 줄 생략 규칙 유지)

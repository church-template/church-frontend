# 🚀 [기능개선][설교][예배] 설교 등록 UX 개선 및 예배 안내 페이지 보강

## 📝 현재 문제점

- 설교 등록 시 설교자 칸이 비어 있어 매번 직접 입력해야 함 (설교자가 바뀌는 일이 거의 없음)
- 설교일 연도 칸에 6자리까지 입력되어, 4자리를 채워도 월로 자동 이동하지 않음
- 설교일이 비어 있어 매번 오늘 날짜를 직접 입력해야 함
- 예배 시간 안내 카드에서 핵심 정보인 예배 시간이 작고 흐리게 표시되어 예배명보다 눈에 띄지 않음
- 예배 안내 페이지에서 설교 목록으로 바로 가는 길이 없어 설교 메뉴를 따로 찾아야 함

## 🛠️ 해결 방안 / 제안 기능

- 설교자 칸에 교회별 기본 설교자가 미리 채워지도록 함 (교회 템플릿이므로 교회별 상수 주입, 수정 화면은 기존 값 유지)
- 설교일 연도 입력을 4자리로 제한하고 4자리 입력 시 월로 자동 이동
- 설교일 기본값으로 오늘 날짜를 미리 채움 (한국 시간 자정 기준으로 날짜 전환)
- 예배 시간 안내 카드의 시간 표기를 더 크고 진하게 강조
- 예배 안내 페이지에 설교 목록으로 가는 진입 버튼 추가

## ⚙️ 작업 내용

- 설교 등록 폼 기본값(설교자·설교일) 적용
- 설교일 입력 연도 4자리 제한
- 예배 카드 시간 강조 타이포 위계 신설·적용
- 예배 안내 페이지 설교 진입 버튼 추가

# 🚀 [기능개선][설교][권한] 설교 목록·상세 회원전용 전환 (SERMON_VIEW)

## 📝 현재 문제점

- 설교 목록·상세가 비로그인 사용자에게 전부 공개되어 있음
- 갤러리는 교인 승인(MEMBER) 후에만 열람 가능한데, 설교는 같은 교인 대상 콘텐츠임에도 접근 제한이 없음
- 회원전용 진입 안내(로그인·교인 승인)가 설교에는 없음

## 🛠️ 해결 방안 / 제안 기능

- 설교 목록·상세를 갤러리와 동일하게 회원전용(교인 승인 후 열람)으로 전환
- 백엔드는 SERMON_VIEW 권한 신설·적용 완료 상태 — 이번 작업은 프론트 전환
- 비로그인 진입 시 로그인 안내, 승인 전 회원(USER)은 교인 승인 안내 표시
- 메인 페이지의 최신 설교 3건은 티저(제목·설교자·날짜만)로 유지, 클릭 시 로그인 유도

## ⚙️ 작업 내용

- 회원전용 게이트를 갤러리·설교 공용 컴포넌트로 일반화하고 갤러리 화면에 적용
- 설교 목록·상세·수정 화면을 회원 영역 데이터 조회 방식으로 전환
- 설교 등록·수정·삭제 후 목록·상세가 즉시 갱신되도록 무효화 방식 전환
- 가이드 문서에 설교 회원전용 정책 반영

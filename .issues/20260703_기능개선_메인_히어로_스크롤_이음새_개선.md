## 제목

🚀 [기능개선][메인][히어로] 십자가 히어로 → 콜라주 스크롤 연출 사진 중복 노출 제거

## 본문

## 📝 현재 문제점

- 메인 홈에서 스크롤 시 십자가 열쇠구멍 모양의 히어로가 확대되며 배경 사진이 선명하게 드러난다.
- 이어서 더 내리면 같은 사진이 한 장 더 풀스크린으로 나타난 뒤 카드 형태로 축소되는데, 동일한 사진이 두 번 연속으로 보이는 이음새 구간이 노출된다.
- 히어로 섹션과 콜라주 섹션이 각각 별개의 스크롤 영역으로 순차 배치되어 있어, 사이에 같은 사진 두 장이 겹쳐 지나가는 어색한 전환이 생긴다.

## 🛠️ 해결 방안 / 제안 기능

- 히어로에서 선명하게 드러난 사진이 그대로 이어져 축소되도록 두 섹션의 전환 이음새를 없앤다.
- 사용자 눈에는 한 장의 사진이 확대되어 선명해진 뒤 그대로 카드로 축소되는 하나의 연속 연출로 보이게 한다.
- 동일 사진이 두 번 반복 노출되는 현상을 제거한다.

## ⚙️ 작업 내용

- 십자가 히어로 마지막 프레임과 콜라주 시작 프레임(동일 사진 풀스크린)이 같은 순간 화면상에서 이어지도록 두 섹션의 스크롤 구간을 겹친다.
- 히어로 카피(문구)가 전환 순간에 갑자기 사라지지 않도록 등장·퇴장 타이밍을 보정한다.
- 데스크톱 스크롤 연출에만 적용하고, 모바일과 모션 최소화(reduced-motion) 환경에서는 기존 정적 표시를 유지한다.

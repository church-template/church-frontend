# ⚙️ [기능추가][차량운행] 차량 탑승 신청·운행 관리 웹 적용

## 📝 현재 문제점

- 백엔드에 차량운행 API(운행일 관리·탑승 신청·탑승 명단)가 추가되었지만 웹에 사용할 화면이 없다
- 교인이 차량 탑승을 신청할 방법이 없고, 운영자도 운행일과 탑승 명단을 웹에서 관리할 수 없다

## 🛠️ 해결 방안 / 제안 기능

- 회원(승인 교인)용 차량 탑승 신청 페이지: 다가오는 운행일 목록에서 픽업 장소·메모를 적어 신청하고, 내 신청 확인·취소
- 운영자용 차량운행 관리 화면: 운행일 등록·수정·삭제와 운행일별 탑승 명단(이름·연락처·픽업 장소) 확인
- 네비게이션 예배·설교 그룹과 마이페이지 관리 허브에 진입점 노출

## ⚙️ 작업 내용

- 회원 페이지 /vehicle-runs: 운행일 목록·탑승 신청·취소 (VEHICLE_APPLY 게이트)
- 어드민 /mypage/manage/vehicle-runs: 운행일 목록·등록·수정·삭제 (VEHICLE_MANAGE)
- 어드민 /mypage/manage/vehicle-runs/[id]: 탑승 명단 조회 (연락처 바로 걸기)
- 네비게이션·관리 허브·권한 라벨 등록

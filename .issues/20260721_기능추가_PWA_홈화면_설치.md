# ⚙️ [기능추가][PWA] 홈 화면 설치 지원 (manifest + 아이콘)

## 📝 현재 문제점

- 홈페이지에 웹 앱 manifest가 없어 홈 화면에 앱으로 설치할 수 없고, 브라우저 탭으로만 접근 가능하다
- iOS에서 홈 화면에 추가해도 아이콘·standalone 실행이 지원되지 않는다

## 🛠️ 해결 방안 / 제안 기능

- 웹 앱 manifest와 설치용 아이콘을 추가해 홈 화면 설치(PWA)를 지원한다
- Android·데스크톱 Chrome은 브라우저 설치 프롬프트가 자동으로 뜨고, iOS는 공유 → 홈 화면에 추가로 앱처럼(standalone) 실행된다
- 서비스워커·오프라인 지원·푸시 알림·설치 안내 UI는 이번 범위에서 제외한다

## ⚙️ 작업 내용

- 웹 앱 manifest 추가 (교회명·설명은 기존 상수에서 주입)
- 설치용 아이콘 4종 생성 (192 / 512 / maskable / apple)
- 설치 가능 판정 검증 (빌드·실서버 응답·DevTools Manifest 확인)

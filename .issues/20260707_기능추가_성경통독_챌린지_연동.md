## 제목

⚙️ [기능추가][성경통독] 성경통독 챌린지 API 연동 및 UI 구현

## 본문

## 📝 현재 문제점

- 백엔드에 성경통독 챌린지 API가 추가되었으나 프론트에는 관련 화면·연동이 전혀 없다.
- 교인이 챌린지를 조회·참여하고 읽기 기록을 남길 수 있는 회원 영역 UI가 없다.
- 관리자가 챌린지를 개설·수정·삭제할 수 있는 어드민 화면이 없다.

## 🛠️ 해결 방안 / 제안 기능

- 추가된 성경통독 챌린지 API를 프론트 UI와 연결한다.
- 공개/회원 조회, 참여, 읽기 기록, 관리자 CRUD까지 도메인 전 흐름을 커버한다.

## ⚙️ 작업 내용

공개/회원 영역
- 챌린지 목록 (GET /api/bible-challenges)
- 챌린지 상세 (GET /api/bible-challenges/{id})
- 챌린지 참여 (POST /api/bible-challenges/{id}/join)
- 내 진행 대시보드 (GET /api/bible-challenges/{id}/my-progress)
- 내 읽기 로그 (GET /api/bible-challenges/{id}/my-logs)
- 읽음 기록 / 취소 (POST·DELETE /api/bible-challenges/{id}/read)
- 내 참여 이력 (GET /api/bible-challenges/my-participations)

관리자 영역
- 챌린지 개설 (POST /api/admin/bible-challenges)
- 챌린지 수정 (PATCH /api/admin/bible-challenges/{id})
- 챌린지 삭제 (DELETE /api/admin/bible-challenges/{id})

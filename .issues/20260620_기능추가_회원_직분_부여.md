## 제목

⚙️ [기능추가][회원관리] 회원에게 직분(position) 부여/변경 기능 추가

## 본문

## 📝 현재 문제점

- 회원 관리 화면에서 역할(role)과 권한(permission)은 부여/회수할 수 있으나, 직분(position)을 회원에게 부여하거나 변경하는 기능이 없다.
- 직분은 직분 관리(PositionManager)에서 이름·정렬순서만 CRUD할 수 있을 뿐, 그 직분을 특정 회원에게 연결하는 경로가 존재하지 않는다.
- 현재 회원의 직분 값은 회원 목록 테이블과 회원 상세 다이얼로그에서 조회 표시만 가능하다.
- 회원 수정 폼(이름·전화·이메일)에도 직분 항목이 없어, 운영자가 회원의 직분을 지정/변경할 수 없다.

## 🛠️ 해결 방안 / 제안 기능

- 회원 상세 다이얼로그에서 회원의 직분을 부여/변경/해제할 수 있는 UI를 추가한다. (역할 부여 영역과 유사한 형태)
- 직분 목록은 직분 관리에서 관리되는 카탈로그를 그대로 선택지로 사용한다.

## ⚙️ 작업 내용

- 프론트엔드: 회원 상세 다이얼로그에 직분 선택/변경 UI 추가

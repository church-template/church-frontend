# 🚀 [기능개선][어드민][에디터] 마크다운 에디터 서식 툴바·사용법 도움말 추가

## 📝 현재 문제점

- 공지·설교·일정 등 본문 작성이 마크다운 문법 기반인데, 마크다운을 모르는 관리자는 서식(제목·굵게·목록 등)을 적용하지 못함
- 본문에 이미지를 넣으려면 미디어 번호 코드를 직접 알아내어 손으로 입력해야 함
- 화면 어디에도 마크다운 사용법 안내가 없음

## 🛠️ 해결 방안 / 제안 기능

- 본문 입력창 위에 항상 보이는 서식 버튼 툴바 제공: 제목(3단계)·굵게·기울임·취소선·글머리 목록·번호 목록·인용·구분선·표
- 링크·유튜브는 주소 입력창으로 받아 자동 삽입, 유튜브 주소는 기존처럼 본문에서 영상으로 표시
- 이미지 버튼은 기존 미디어 선택창과 연동 — 사진을 고르면 본문에 자동 삽입되어 코드를 몰라도 됨
- 도움말 버튼: "이렇게 쓰면 → 이렇게 보입니다" 형식의 사용법 안내창 제공
- 공지·설교·일정·앨범·챌린지·부서 본문 입력 모두에 동일 적용

## ⚙️ 작업 내용

- 이미지 선택창 연동 삽입 추가

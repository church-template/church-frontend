## 제목

⚙️ [기능추가][어드민][부서] 부서 계층 관리

## 본문

## 📝 현재 문제점

- 부서 계층을 운영자가 생성·수정·삭제할 화면이 없다.
- 공개 부서 소개 페이지는 별도 상수로 구동되어 관리용 부서 데이터와 분리되어 있어, 인라인으로 얹을 공개 화면이 없다.

## 🛠️ 해결 방안 / 제안 기능

- 마이페이지 관리 허브 하위 전용 화면에서 부서 계층을 생성·수정·삭제한다.
- 이 화면의 변경은 공개 부서 소개에 자동 반영되지 않으며, 두 데이터가 별개임을 화면에 안내한다.

## ⚙️ 작업 내용

- 부서 목록을 계층(트리)으로 표시하고 행별로 수정·삭제
- 부서 생성·수정 (이름, 설명, 담당자, 상위 부서, 정렬 순서)
- 상위 부서 변경 시 자기 자신·자기 하위 선택을 막아 순환 방지
- 하위 부서가 있으면 삭제 차단
- 루트로 변경과 단순 수정 경로 구분
- 동시 수정 충돌 안내
- 공개 소개 페이지와 별개 데이터라는 안내 문구

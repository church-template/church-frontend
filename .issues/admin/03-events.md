## 제목

⚙️ [기능추가][어드민][일정] 일정 등록·수정·삭제

## 본문

## 📝 현재 문제점

- 일정을 운영자가 등록·수정·삭제할 화면이 없다. 백엔드 API는 준비되어 있으나 프론트가 연동되어 있지 않다.

## 🛠️ 해결 방안 / 제안 기능

- 공개 일정 캘린더와 일정 상세에 권한 보유 시 등록·수정·삭제 액션을 인라인으로 노출한다.
- 본문이 짧은 도메인이라 별도 화면 대신 팝업 폼으로 입력한다.

## ⚙️ 작업 내용

- 일정 등록·수정·삭제 (캘린더와 상세에서, 권한 보유 시 노출)
- 시작·종료 시간, 종일 토글, 장소, 태그 입력
- 종료가 시작보다 이후인지 검증
- 동시 수정 충돌 안내
- 날짜·시간을 한국 시간 기준으로 일관되게 표시

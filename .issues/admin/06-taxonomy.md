## 제목

⚙️ [기능추가][어드민][태그·직분] 분류·운영 항목 관리

## 본문

## 📝 현재 문제점

- 전역 태그와 직분을 운영자가 관리할 화면이 없다. 두 항목 모두 공개 페이지가 없는 운영 데이터다.

## 🛠️ 해결 방안 / 제안 기능

- 관리 허브 하위 전용 화면에서 이름 중심의 간단한 추가·수정·삭제를 제공한다.
- 태그와 직분은 권한이 독립이라 보유한 권한에 해당하는 화면만 보인다.

## ⚙️ 작업 내용

- 태그 관리 (추가·수정·삭제). 삭제 시 연결된 콘텐츠에서 태그가 함께 제거된다는 경고
- 직분 관리 (추가·수정·삭제, 정렬 순서)
- 직분은 표시용 이름이며 권한과 무관하다는 안내
- 같은 이름 중복 시 안내

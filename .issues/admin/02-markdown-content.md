## 본문

## 📝 현재 문제점

- 설교와 공지를 운영자가 등록·수정·삭제할 화면이 없다. 백엔드 API는 준비되어 있으나 프론트가 연동되어 있지 않다.

## 🛠️ 해결 방안 / 제안 기능

- 공개 설교·공지 목록과 상세에 권한 보유 시 등록·수정·삭제 액션을 노출한다.
- 본문이 길어 작성·수정은 전용 편집 화면을 쓰고, 본문은 작성/미리보기 탭으로 입력하며 태그를 함께 지정한다.

## ⚙️ 작업 내용

- 설교 등록·수정·삭제 (전용 편집 화면, 권한 보유 시 노출)
- 공지 등록·수정·삭제, 상단 고정 토글
- 본문 작성/미리보기 에디터, 본문에 이미지·PDF 첨부
- 태그 다중 선택
- 동시 수정 충돌 안내, 입력값 검증과 오류 처리

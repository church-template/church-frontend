## 본문

## 📝 현재 문제점

- 회원, 역할, 권한, 약관을 운영자가 관리할 화면이 없다. 가장 민감한 영역이라 신중한 처리가 필요하다.

## 🛠️ 해결 방안 / 제안 기능

- 관리 허브 하위 전용 화면에서 회원 목록·검색·상세, 역할과 권한 매트릭스, 회원 역할 부여·회수, 비밀번호 초기화, 약관 일괄 리셋을 제공한다.
- 상위 권한·시스템 역할·자기 자신에 대한 변경은 미리 막아 실수와 권한 상승을 방지한다.

## ⚙️ 작업 내용

- 회원 목록·검색·상세 조회, 회원 정보 수정
- 회원 역할 부여·회수 (역할 부여가 곧 교인 승인)
- 비밀번호 초기화 (임시 비밀번호 1회 표시, 화면 외 잔류 금지)
- 역할 추가·수정·삭제, 역할별 권한 일괄 설정
- 약관 일괄 리셋 (대상 선택, 재동의 유도, 강한 확인창)
- 권한 위계 가드 (상위 권한·시스템 역할·자기 자신은 버튼 비활성)

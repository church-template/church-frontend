## 제목

⚙️ [기능추가][어드민][미디어] 갤러리·주보·미디어 라이브러리 관리

## 본문

## 📝 현재 문제점

- 갤러리 앨범·사진, 주보, 미디어 파일을 운영자가 올리고 관리할 화면이 없다. 파일 업로드가 핵심인 영역인데 연동되어 있지 않다.

## 🛠️ 해결 방안 / 제안 기능

- 미디어 라이브러리는 관리 허브 하위 전용 화면으로 두고, 갤러리는 상세 화면에 인라인 관리, 주보는 목록에 인라인 등록을 붙인다.
- 업로드한 파일은 라이브러리에 모이고 여러 콘텐츠가 재사용한다.

## ⚙️ 작업 내용

- 미디어 라이브러리 전용 화면 (업로드, 목록, 삭제)
- 미디어 삭제 시 사용 중이면 사용처를 보여주고 삭제 차단
- 갤러리 앨범 생성·수정·삭제, 사진 다중 업로드 (신규 업로드와 기존 파일 재사용 혼합)
- 주보 등록·수정·삭제 (PDF 업로드 또는 기존 파일 재사용)
- 동시 수정 충돌 안내
- 허용 형식과 용량 초과 안내

# 부서 상세 보강 섹션 확대 적용 (#82)

### 📌 작업 개요
학생부에만 적용돼 있던 부서 상세 "보강 섹션"(소개·특징·안내·주요 활동·특별 프로그램·활동 사진·초대) 구조를 나머지 4개 부서(청년부·예배부·남선교회·여선교회)로 확대 적용. 각 부서 콘텐츠와 활동 사진을 상수에 주입하고, 모바일 네비게이션의 스크롤·로그인 버튼 위치를 함께 정리.

### 🎯 구현 목표
- 부서 상세 페이지 콘텐츠 구조를 전 부서로 통일(학생부와 동일 포맷)
- 콘텐츠는 하드코딩 금지 원칙에 맞춰 `departments.ts` 상수로 주입
- 부서별 활동 사진 4장씩 정적 에셋 추가
- 모바일 시트 네비 스크롤/로그인 버튼 위치 개선

### ✅ 구현 내용

#### 부서 상세 콘텐츠 확대 (청년부·예배부·남선교회·여선교회)
- **파일**: `src/constants/departments.ts` (+148)
- **변경 내용**: 4개 부서 항목에 `intro`(소개), `features`(특징 3), `info`(담당·시간·연락처·장소), `activities`(주요 활동), `programs`(특별 프로그램 4), `gallery`(활동 사진 4), `invite`(초대 문구) 필드 추가
- **이유**: 학생부에만 있던 상세 섹션 구조를 전 부서로 통일해 상세 페이지 UX를 일관되게 맞춤. 텍스트·이미지 모두 상수 주입으로 콘텐츠 하드코딩 금지 규칙 준수

#### 부서 활동 사진 에셋 추가
- **파일**: `public/dept/youth/*`, `public/dept/praise/*`, `public/dept/men/*`, `public/dept/women/*` (각 1~4.jpg)
- **변경 내용**: 부서별 활동 사진 4장씩 정적 에셋 배치, `gallery` 필드에서 참조
- **이유**: 활동 사진 섹션을 상세 페이지 하단에 노출하기 위한 이미지 소스 확보

#### 모바일 네비게이션 스크롤·로그인 버튼 위치 수정
- **파일**: `src/components/shell/MobileNav.tsx` (9/9), `src/components/ui/sheet.tsx` (1/1)
- **변경 내용**:
  - 로그인 버튼을 시트 하단(`mt-auto`)에서 **상단**으로 이동, 하단 헤어라인 → 상단 헤어라인 구분
  - 네비 링크 타이포 상향: 1뎁스 `navLink → titleMd`, 하위 `bodySm → bodyMd` (고령 사용자 가시성)
  - `SheetContent`에 `overflow-y-auto overscroll-contain` 추가 → 메뉴가 길어져도 시트 내부 스크롤
- **이유**: 메뉴 항목이 늘면서 하단 고정 로그인 버튼이 화면 밖으로 밀리는 문제 해결. 시트 자체 스크롤로 전 메뉴 접근 보장

### 📌 참고사항
- 콘텐츠 텍스트·이미지는 전량 `departments.ts` 상수 주입 (하드코딩 없음)
- 부서 상세 섹션 렌더 순서는 이전 세션의 재배치안(소개 → 안내 → 주요 활동 → 특별 프로그램 → 활동 사진)을 따름
- 구현 계획 문서: `docs/superpowers/plans/2026-07-03-dept-sections-rollout.md`

### 📌 작업 개요

`/about/history` 연혁 페이지를 카카오 지속가능성 사이트 스타일의 **에디토리얼 헤어라인 그리드**로 전면 개편. 좌측 sticky 사진 + 우측 카드 2단 구조를 폐기하고, 시대(챕터)마다 대형 번호 + 연도·제목 + 사진 + 본문이 헤어라인 셀에 담기는 지그재그 그리드로 재구성. 마지막 챕터는 다크 밴드로 마무리하고, 사진 셀에 방향성 와이프 연출을 추가.

**보고서 파일**: `.report/20260705_#86_연혁_에디토리얼_그리드_개편.md`

### 🎯 구현 목표

- 좌 sticky 2단 → 카카오식 풀폭 헤어라인 그리드 챕터로 전환
- 챕터마다 초대형 번호(01~07) + 좌우 미러(지그재그) 배치
- 마지막 시대(2025 디지털 사역)를 다크 밴드로 톤 반전 (페이지 리듬)
- 뷰포트 중앙 카드 추적용 스크롤 엔진 제거 (구조 단순화)
- 사진 셀 방향성 와이프 리빌 (사진 위치에 따라 좌→우 / 우→좌)

### ✅ 구현 내용

#### 챕터 그리드 컴포넌트 재작성
- **파일**: `src/components/history/HistoryChapter.tsx`
- **변경 내용**: `forwardRef` 제거, CSS Grid 셀 마크업으로 재작성. props에 `index`(번호·미러 방향 파생), `dark`(마지막 챕터 여부) 추가
- **이유**: 번호는 `index + 1`을 2자리 패딩("01"~"07")으로 파생, 홀짝으로 미러 방향 결정. 스크롤 측정 대상이 아니게 되어 ref 불필요

#### 스토리 컨테이너 단순화
- **파일**: `src/components/history/HistoryStory.tsx`
- **변경 내용**: 좌측 sticky aside와 스크롤 엔진 훅 사용 제거. 히어로 + 챕터 `<ol>` 렌더만 유지, 마지막 챕터에 `dark` 전달
- **이유**: 풀폭 그리드는 활성 시대 추적이 불필요 → 구조·코드 대폭 축소

#### 스크롤 엔진 삭제
- **파일**: `src/components/history/useHistoryScrollEngine.ts`(+테스트) **삭제**
- **이유**: 좌측 sticky 사진 동기화 전용이었으나 sticky 구조가 사라져 사용처 소멸

#### 그리드 스타일 재작성
- **파일**: `src/components/history/HistoryStory.module.css`
- **변경 내용**: "컨테이너 배경(헤어라인색) + gap 1px" 트릭으로 셀 구분선 구현. 데스크톱은 `grid-template-areas`로 표준/미러 방향 전환, 모바일은 세로 스택
- **이유**: 셀별 보더 방향 관리 없이 미러·모바일에서 구분선 자동 정합. 외곽만 `rounded-xl`(24px)로 "직각 금지" 규칙과 그리드 인상 양립

#### 사진 셀 방향성 와이프 리빌
- **파일**: `src/components/main/Reveal.tsx`, `src/components/history/HistoryStory.module.css`
- **변경 내용**: `Reveal` 등장 시 `data-revealed` 속성을 추가로 부여. 사진 셀은 이 신호와 동기해 `clip-path: inset` 전환 — 사진 우측(표준)은 좌→우, 좌측(미러·데스크톱)은 우→좌
- **이유**: CSS 모듈 해시 클래스는 외부 CSS가 선택 불가 → 하위 모듈이 등장과 동기된 연출을 걸 수 있는 공개 신호(`data-revealed`)가 필요

### 🔧 주요 변경사항 상세

#### 다크 챕터 구분선 (color-mix 보정)
초기 구현은 `rgba(255,255,255,0.16)`을 다크 셀 구분선에 사용했으나, gap 라인이 다크 셀이 아닌 **흰 페이지 배경 위에 합성**되어 16% 알파가 무효화(= 순수 흰색)되는 문제가 있었음. `color-mix(in srgb, var(--color-on-dark) 16%, var(--color-surface-dark))`로 교체하여 토큰 혼합으로 산출 — 실측 배경색 ≈ rgb(49, 50, 52)의 은은한 헤어라인으로 렌더 확인.

#### 챕터 번호 tnum
챕터 번호에 `tabular-nums` 적용하여 미러 챕터 우측 정렬 시 자릿수 폭 흔들림 방지.

**특이사항**:
- 번호는 장식이므로 `aria-hidden` — 순서 의미는 연도 텍스트가 담당
- 미러는 `grid-template-areas`만 뒤집고 DOM 순서는 유지 → 읽기 순서·접근성 보존
- 데이터(`HISTORY` 상수)·메인 `HistoryBand`는 무변경 (범위: `/about/history`만)
- 와이프·다크 밴드 모두 `prefers-reduced-motion`에서 정적 표시
